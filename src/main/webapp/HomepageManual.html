Help






-On the bottom-left of the homepage, there is a button called 'choose zip file'.
When clicked, this button allows you to upload a 'state.zip' file to the website.

-On the bottom-right of the homepage, there is a button called 'log out'.
When clicked, this button logs you out of the homepage.

Selecting a graph is done by a combination of 3 buttons:
-The choose a graph type button allows you to choose what type of chart you want.
The possibilities are: PieChart, BarChart, LineChart.

-The select a lane button allows you to choose between different lanes
in order to visualize different parts of the data.

-Finally, to actually generate the graph, press the generate graph button.
